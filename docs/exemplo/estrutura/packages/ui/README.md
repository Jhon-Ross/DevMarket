# packages/ui — Design System

Responsabilidades
- Componentes presentational: sem chamadas de dados e sem acoplar ao app.
- Uso de tokens semânticos (CSS variables) para cores/spacing/tipografia.

Decisões
- Composição sobre herança; foco em acessibilidade (Radix UI como base opcional).
- Núcleo inicial: `Button`, `Card`, `Avatar`, `Tag`, `Grid`, `MediaGallery`.
