# Pasta de Exemplo — Base do Projeto

Esta pasta demonstra:
- Estrutura de pastas base (apps, packages, prisma).
- Backlog pré–Sprint 0 (visual + estrutura) com critérios de aceitação.
- Board Kanban para acompanhar execução no Obsidian.

Arquivos úteis
- Estrutura (árvore): `./estrutura/tree.txt`
- Descrição por área:
  - `./estrutura/apps/web/README.md`
  - `./estrutura/apps/studio/README.md`
  - `./estrutura/packages/ui/README.md`
  - `./estrutura/packages/sanity/README.md`
  - `./estrutura/packages/lib/README.md`
  - `./estrutura/packages/types/README.md`
  - `./estrutura/packages/config/README.md`
  - `./estrutura/prisma/README.md`
- Backlog: `./backlog/pre-sprint0-visual-estrutura.md`
- Kanban: `./kanban/pre-sprint0-visual-estrutura.md`

Como usar
- Leia `tree.txt` para entender a visão geral.
- Abra o backlog e mova os cards no Kanban conforme avança.
